# Mario 1-1 in 3D

A faithful 3D recreation of Super Mario Bros World 1-1, built with Three.js.
Playable in the browser on desktop and mobile (on-screen touch controls).

**Play:** open the GitHub Pages URL on your phone, tap to start.
**Controls:** ◀ ▶ to move, JUMP to jump (or arrow keys + space on desktop).

Static site — no build step. Source lives in `public/`.

Environment props (ground blocks) generated via the Thrixel MCP; characters use
styled primitives (Thrixel doesn't do recognizable characters). See git history.
